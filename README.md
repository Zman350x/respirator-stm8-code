# Respirator STM8 Code
## Description

Firmware for the STM8S003F3 chip that controls the respirator. Made for Intro
Engineering Design final project, Fall 2025.

Corresponding PCB can be found
[here](https://github.com/Zman350x/respirator-main-board).

## Tools

Made with:
* [SDCC](https://sdcc.sourceforge.net)
* [stm8flash](https://github.com/vdudouyt/stm8flash)
* [stm8-binutils-gdb](https://stm8-binutils-gdb.sourceforge.io)
* [OpenOCD](https://openocd.org)
